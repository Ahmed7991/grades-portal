Student Grade PortalThis is a secure, private grade distribution app built with Streamlit.How to Deploy (Put it online)Create a GitHub Account: Go to github.com and sign up.Create a Repository: Click the "+" icon and select "New Repository". Name it grade-portal (or anything you like).Upload Files: Upload the following 3 files to your new repository:app.pygrades.csvrequirements.txtDeploy on Streamlit:Go to share.streamlit.io and sign up with your GitHub account.Click "New App".Select your grade-portal repository from the dropdown.Click "Deploy".Updating GradesTo update grades or add new students, simply edit the grades.csv file on GitHub (or upload a new version), and the website will update automatically.Security NoteThis app uses a "Server-Side" check. This means the full list of grades (grades.csv) stays on the server. The student's browser only receives the single grade associated with the key they entered.
